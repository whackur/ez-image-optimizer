# 사용방법

### CLI mode

```bash
usage: ez-image-optimizer.py [-h] [-f F]

optional arguments:
  -h, --help  show this help message and exit
  -f F        Specific Image file path to convert.
              Basically, If do not use it,
              [./img/*] images will be optimized in [./img-optimized]
              or change conf.py
```
